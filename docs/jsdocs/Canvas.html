Class: Canvas

Canvas(el)

This is the main Canvas class. This class implement a general purpose canvas that can manage nodes, components, sockets (a special subset of components) and connections between sockets. The canvas is able to zoom and pan itself as well as to move components. Components are organized in a hierarchical way, so that moving a component will move all its subcomponents, including sockets. Sockets are (sub)components that can be connected through connections to other sockets. The class use SVG elements to render all elements.

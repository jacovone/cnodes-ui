Class: Component

Component()

This class is the base class for all components in the cnodes-ui canvas. The class is responsible for manage mobility of components. This is not yet the class for the cnodes nodes, is more abstract so that the user can derive from that to implement components that are not explicitly connected to the cnodes library, such as decorators, comments, etc. In general component can have subcomponents. Special examples of subcomponents are sockets. The socket is a component that can be connected through connections, in the canvas-ui class model.
